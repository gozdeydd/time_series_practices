# Time Series Analysis

In this repository I share practices my Time series practices in Python using sktime, tbats, darts and prophet.




The purpose of this repository is to give ideas to the beginners about time series analysis. There are 3 projects in this repository

1. Apple Project : This project trying to analyze price change onstock prices of Apple specifically trying to capture Covid effect.

2. Stackoverflow : This project is trying to analyze change in the Stackoverflow rates of 'Matlab' and its change overtime. 

3. Toscos: This dataset is belong to the a daily data of a market chain of 10 shops in 3 different regions. Data have the information of weekends and bank holidays. 

Notebooks use the corresponding data that is mentioned in the notebook. Environement should be  loaded to local from requirements file by below command:


'pip install requirements.txt'
